üì¶ Quick Setup

Get Started in Seconds

Choose your preferred installation method and start building amazing select components

Install via Package Manager

Using npm:

npm install select-library

Using yarn:

yarn add select-library
Include via CDN

Add these tags to your HTML:

<!-- CSS -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/select-library@3.0.0/dist/select.min.css">

<!-- JavaScript -->
<script src="https://cdn.jsdelivr.net/npm/select-library@3.0.0/dist/select.min.js"></script>
Basic Usage

Initialize Select on any HTML select element:

import Select from 'select-library';
import 'select-library/dist/select.css';

// Basic usage
const mySelect = new Select('#my-select', {
  searchable: true,
  multiple: true,
  placeholder: 'Choose options...'
});

// With options
const advancedSelect = new Select('#advanced-select', {
  searchable: true,
  multiple: true,
  maxSelections: 5,
  closeOnSelect: false,
  searchPlaceholder: 'Type to search...'
});
That's it! Your select component is now enhanced with search, multi-select, and modern styling.
